Description

Close a list of tabs, or close all but a list of tabs.


Config

action enum
If the action is set to Close, the node will close all the tabs specified. If set to Keep, the node will close all tabs except the ones specified.
timeout number
The time (in milliseconds) within which if the action doesn't complete, the node errors out. This property is not functional right now.
tabIds array
Array of tab ids or tab objects to close or keep (according to value of action)
session config
Holds the encryption keys used by Maya to securely talk to your browser.

Outputs

This node does not set any properties on the msg object in case of successful execution.


On error

The node will set the below-mentioned properties on the msg object, when there is an error

isError boolean
Set to true if the close operation failed for whatever reason. This property might not be set at all in case of no error.
error object
Contains information about the error that occured

Description

Modify properties such as URL (can be used for redirection), mute status, hidden status, etc. on a list of tabs


Config

timeout number
The time (in milliseconds) within which if the action doesn't complete, the node errors out. This property is not used right now, and setting it will make no difference.
updatesobject
The JSON object that specifies the modifications to be made to the tab. The structure of this object is documented in detail here. Properties not specified in this object, will remain unmodified on the tab.
tabId array
The unique tab id in which the website to be queried is rendered in. It can be obtained in using {"url":"*://*.[domain]/*"} in the find tab node. For e.g. {"url":"*://*.[domain]/*"} where domain can be anything like google.com or amazon.in. There are other query elements which can be added to same object as referenced above. One common one to be used is for activeTab which takes in a boolean to search for a tab in focus with the url pattern matching above.
session config
Holds the encryption keys used by Maya to securely talk to your browser.

Outputs

The following properties are set on the msg object in case of successful execution.

updatedTabsarray
Array of browser tab objects representing all the tabs that were updated. You can see what properties a tab object contains in the Google Chrome docs.

On error

The node will set the below-mentioned properties on the msg object, when there is an error

isErrorboolean
Set to true if the operation failed for whatever reason. Note that this property might not be set at all in case of no error.
error object
Contains information about the error that occured

Description

Create, delete and query browser bookmarks and bookmark folders

Config

func enum
What you want to do with the node. If set to "getBookmarkFolders", node will return list of bookmark folders. If set to "getBookmark", node will get all bookmarks from a particular folder with id specified in "parentId" (or all folders if the field is set to -1). If set to "addBookmark" it will add a bookmark to the folder with id "parentId", with all other fields with their obvious meaning. If set to "removeBookmark", the node will delete the bookmark with id specified by "Bookmark ID".

Outputs

bookmarks various
Present on message if "func" is equal to "getBookmark". Contains list of bookmark objects.
bookmarkId string
Present on message if "func" is equal to "addBookmark" or "deleteBookmark". Is equal to the ID of the bookmark added or deleted respectively.
bookmarkFolders various
Present on message if "func" is equal to "getBookmarkFolders". Contains list of bookmark-folder objects.

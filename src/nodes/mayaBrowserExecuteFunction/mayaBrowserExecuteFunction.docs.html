Description

Execute a given function with given arguments, on a given element on the page. Using this node on a page with it's arguments will have the same effect as running $x(selector)[0][function](...arguments) in the browser console of that page.


Config

selectorstring
The xpath for the element to execute the function on. We recommend watching this video to learn more about xpaths (it's slightly long, but its worth it).
timeout number
The amount of time in milliseconds that the node will wait for the target element to load in page. Whenever the element is loaded, the given function is executed immediately. If it doesn't load within this time period, the node will error out.
tabId number
The id of the tab inside which this action is to be performed.
function string
The name of the function to be called on the target element.
arguments array
Array of positional arguments to be passed to the function as arguments.
session config
Holds the encryption keys used by Maya to securely talk to your browser.

Outputs

This node does not set any properties on the msg object in case of successful execution.


On error

The node will set the below-mentioned properties on the msg object, when there is an error

isErrorboolean
Set to true if the operation failed for whatever reason. This property might not be set at all in case of no error.
error object
Contains information about the error that occured

Description

Open a URL in a new browser tab.


Config

urlstring
The URL to open.
session config
Holds the encryption keys used by Maya to securely talk to your browser.

Outputs

The following properties are set on the msg object in case of successful execution.

tabsarray
Array containing a single browser tab object representing the new tab that was just opened (i.e., you can access it through msg.tabs[0]). You can see what this tab object contains in the Google Chrome docs.

On error

The node will set the below-mentioned properties on the msg object, when there is an error

isErrorboolean
Set to true if the operation failed for whatever reason. Note that this property might not be set at all in case of no error.
error object
Contains information about the error that occured
